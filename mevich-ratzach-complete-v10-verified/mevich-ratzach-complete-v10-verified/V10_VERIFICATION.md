## UX
- אינדיקטור ברור: כמה קלפים יש ביד.

- “דקה חוזר אליך אני מסיים לאכול”
- “רגע אני יחף”
- “אמא שלי לא מרשה”
- “זה לא שלי נשבע”

## מה לבדוק בהרצה
1. npm install
2. npm run build
3. npm run dev
4. לפתוח חדר
5. להיכנס לתור שחקן
6. לוודא שרואים:
   - 🃏 X קלפים ביד
   - “יש לך X קלפים ביד”
7. לוודא שהיד יורדת ל־6 לרגע וחוזרת ל־7 אחרי בחירה.
